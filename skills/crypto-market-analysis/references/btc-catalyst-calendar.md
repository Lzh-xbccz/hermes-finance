# BTC 催化剂日历模板

> 从 financial-services/catalyst-calendar 适配，专门为 BTC 日内交易者设计。

## 固定宏观事件（每月）

| 日期规律 | 事件 | 对 BTC 影响 |
|---------|------|-----------|
| 每月第一个周五 | 美国非农 (NFP) | HIGH - 影响美元+风险偏好 |
| 每月 10-14 日 | 美国 CPI | HIGH - 通胀预期直接影响 Fed 路径 |
| 每月 13-16 日 | 美国 PPI | MED - 管道通胀 |
| 每月最后一个周五 | 美国 PCE | HIGH - Fed 最关注的通胀指标 |
| 每 6 周 | FOMC 利率决议 | CRITICAL - 利率路径直接决定 BTC 中期方向 |
| 每月末 | 美国 GDP (季度) | MED - 经济增长预期 |

## 加密特有事件（不定期）

| 事件类型 | 示例 | 影响 |
|---------|------|------|
| 美国 BTC 战略储备更新 | White House adviser 声明 | HIGH - 供需基本面 |
| SEC 监管公告 | ETF 审批/拒绝、交易所执法 | HIGH - 市场结构 |
| 比特币 ETF 流量异常 | GBTC/IBIT/FBTC 大额流入/流出 | MED - 机构需求 |
| 比特币减半 | 约每 4 年一次（下次 2028） | CRITICAL - 供给冲击 |
| 以太坊升级 | Pectra 等 | LOW - 间接通过 ETH-BTC |
| 交易所黑客/破产 | FTX 级别事件 | CRITICAL - 市场信心 |
| 稳定币监管 | US 立法进展 | MED - 市场结构 |

## 每周例行检查

分析 BTC 前，检查本周是否处于以下窗口：

- [ ] 本周有 FOMC？（48h 内 → 宏观权重大于技术面）
- [ ] 本周有 CPI/NFP？（24h 内 → 标注事件风险）
- [ ] 本周有重大加密事件？（BTC 储备、ETF 决策、监管公告）
- [ ] 本周有 OPEC 会议？（间接通过油价→通胀预期）
- [ ] 本周是否长假前？（流动性下降 → 方向降级）

## 催化剂对方向的影响判断

| 场景 | 方向偏向 |
|------|---------|
| CPI 低于预期 + FOMC 偏鸽 | 🟢 BTC 看涨 |
| CPI 高于预期 + FOMC 偏鹰 | 🔴 BTC 看跌 |
| 非农强劲 + 利率预期上升 | 🔴 BTC 承压 |
| 非农疲软 + 降息预期升温 | 🟢 BTC 受益 |
| BTC 储备更新正面消息 | 🟢 BTC 看涨 |
| 交易所遭 SEC 执法 | 🔴 BTC 短期看跌 |
