# 商品期货分析适配 (Commodity Adaptation)

> 当用户要求分析原油(CL)、黄金(GC)、天然气(NG)等商品期货时使用。
> 核心思路沿用 crypto-market-analysis 的因果拆解和多维交叉验证，但商品期货必须使用 futures-market-analysis 的七维证据采集 + 第八维 CZSC 确认框架，数据源和指标完全替换。

## 数据源映射

| 加密维度 | 商品等价物 | 数据源 |
|---------|-----------|--------|
| 实时行情 | Yahoo Finance `CL=F`, `GC=F` 等 | `query1.finance.yahoo.com/v8/finance/chart/{SYMBOL}?interval=1d&range=1d` |
| 30日走势 | 同上 `range=90d` | 日线 OHLC |
| 4H K线 | 1H 数据聚合成 4H | `interval=1h&range=5d` → 每 4 根聚合 |
| 1H K线 | 直接获取 | `interval=1h&range=2d` |
| OI + 资金费率 | ❌ 商品期货无永续合约等价物 | 用 OVX (原油恐慌指数) + USO ETF 替代 |
| 多空比 | ❌ 无 | COT 报告滞后 3 天，日内不可用 |
| 链上数据 | ❌ 不适用 | 跳过 |
| 交易所交叉验证 | 单一交易所 (NYMEX/CME) | ⚠️ 无多所对比，标注「单交易所风险」 |
| 恐惧贪婪 | N/A | 用 OVX 作为恐慌指标替代 |
| 美股联动 | 保留 | ^GSPC, ^IXIC, ^DJI, ^VIX |
| 新闻 | Google News RSS | 替换关键词为原油/地缘相关 |

## OVX (原油波动率指数) 解读

OVX 是商品期货分析中**最重要的单一指标**，替代加密的 OI+费率组合：

| OVX 范围 | 市场状态 | 交易含义 |
|---------|---------|---------|
| < 25 | 低波动，安逸 | 可按技术面正常交易 |
| 25-35 | 正常 | 正常仓位 |
| 35-45 | 偏高，供应担忧 | 仓位减半 |
| 45-75 | 地缘危机 | 仓位降至 1/4，止损放宽 |
| > 75 | **极端恐慌/战争模式** | 建议观望。日均预期波幅 = OVX / √252 |

## CL (WTI 原油) 专用参数

SYMBOL = "CL=F"          # Yahoo Finance 代码

## 4H K线聚合方法

Yahoo Finance 不提供 4H K线，从 1H 聚合：

```python
candles = []
for i in range(len(idx)):
    o=quotes['open'][i]; h=quotes['high'][i]
    l=quotes['low'][i]; c=quotes['close'][i]; v=quotes['volume'][i]
    if o and c: candles.append({'t':idx[i],'o':o,'h':h,'l':l,'c':c,'v':v})

agg_4h = []
for i in range(0, len(candles)-3, 4):
    chunk = candles[i:i+4]
    agg_4h.append({
        't': chunk[0]['t'],
        'o': chunk[0]['o'],
        'h': max(x['h'] for x in chunk),
        'l': min(x['l'] for x in chunk),
        'c': chunk[-1]['c'],
        'v': sum(x['v'] for x in chunk)
    })
```

## 决策框架调整

商品期货与加密货币的关键差异：

| 维度 | 加密 | 商品期货 |
|------|------|---------|
| 驱动因素 | 庄家合力 + 技术面 | **地缘政治 > 库存数据 > 美元 > 技术面** |
| 技术分析有效性 | 高 | 低（地缘事件可瞬间覆盖所有技术位） |
| 止损可靠性 | 较高 | 低（跳空常见，尤其周末后开盘） |
| 交易时段 | 24/7 | 美东时间主力时段，其余低流动性 |
| 决策权重 | 庄家行为 40% + 结构 40% + 新闻 20% | **新闻/地缘 60% + OVX 20% + 结构 20%** |

## 新闻解读优先级

商品期货新闻必须按事件等级分类：

| 等级 | 事件类型 | 油价预期反应 | 示例 |
|------|---------|------------|------|
| 🚨 一级 | 海峡关闭/全面战争 | ±$15-20 跳空 | 霍尔木兹海峡正式封锁 |
| 🔴 二级 | 军事冲突升级/停火破裂 | ±$5-10 | 伊朗攻击美军舰、阿联酋遭袭 |
| 🟡 三级 | 停火谈判/库存数据 | ±$3-5 | EIA 库存、OPEC 会议 |
| ⚪ 四级 | 常规供需/评论 | ±$1-2 | 分析师预测、常规报告 |

一级和二级事件面前，**技术面完全无效**。

## 做空/做多条件检查清单

当用户问「做空/做多怎么样」时，用清单逐条回应：

做空需要同时满足：
- [ ] lower high 确认（最近摆高 < 前一摆高）
- [ ] OVX 下降趋势（恐慌溢价消退）
- [ ] 新闻面无新的升级信号
- [ ] 价格跌破关键支撑（非反弹受阻）

做多需要同时满足：
- [ ] higher low 确认
- [ ] OVX 至少不上升
- [ ] 无重大利空新闻
- [ ] 盈亏比 > 1.5（在合理入场点）
