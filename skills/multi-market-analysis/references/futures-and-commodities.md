# Futures and Commodities

用于商品期货、股指期货，以及可以按“宏观驱动 + 结构位”理解的其他期货标的。

## 1. 总原则

1. 先确认是 `期货`、`现货`、`ETF` 还是 `现金指数`。
2. 如果只拿到 `CL=F`、`GC=F` 这类近月代理，要写明这不是用户券商里的精确合约月份。
3. 先看事件和波动率，再看技术图形。
4. 重大事件窗口前，宁可 `观望`，不要在噪音最大的时候硬给方向。

## 2. 权重框架

| 子市场 | 驱动排序 |
|-------|---------|
| 原油 | 地缘/航运 > OPEC+ > 库存 > 美元 > 结构 |
| 黄金 | 实际利率/美债收益率 > 美元 > Fed 预期 > 避险需求 > 结构 |
| 白银 | 黄金方向 > 工业属性 > 美元 > 结构 |
| 铜 | 中国需求/政策 > 库存 > 美元 > 风险偏好 > 结构 |
| 天然气 | 天气/库存 > 产量/LNG > 季节性 > 结构 |
| 股指期货 | 利率/Fed > 财报/权重股 > 风险偏好 > 波动率 > 结构 |

## 3. 原油 `CL` / 能源

核心驱动：
- 地缘风险、海运中断、产油国冲突
- `OPEC+` 配额和口径变化
- `API / EIA` 库存
- 炼厂开工、汽油裂解、需求担忧
- `DXY`

核心风险代理：
- `^OVX` 优先
- 如果 `OVX` 不可得，至少补 `VIX + DXY`

执行规则：
- 一级/二级地缘事件前，技术面权重显著下降
- `EIA`、`OPEC+`、重大停火/升级消息前后，不追单
- 回答里要明确写出当前是“风险溢价抬升”还是“风险溢价回吐”

## 4. 黄金 `GC` / 贵金属

核心驱动：
- `TNX` 或更广义的实际利率方向
- `DXY`
- Fed 降息/鹰派预期
- 风险事件、央行购金、ETF 资金流

执行规则：
- 如果收益率和美元同向走强，黄金多头要降级
- 如果地缘与利率信号冲突，优先写出谁是主导因子
- 金银同看时，白银弹性更大但噪音也更大

## 5. 铜 `HG` / 工业品

核心驱动：
- 中国地产、基建、信用和工业数据
- 交易所库存变化
- 美元强弱
- 风险偏好与全球制造业预期

执行规则：
- 没有中国需求线索时，不要把单日拉升直接解释成趋势反转
- 工业品更容易被宏观情绪带节奏，结构信号需要和基本面同向

## 6. 天然气 `NG`

核心驱动：
- `EIA` 天然气库存
- 气温模型、飓风、供给中断
- LNG 出口节奏

执行规则：
- 波动极大，结构位容易被刺穿
- 如果没有天气或库存背景，只能给低置信度判断

## 7. 股指期货 `ES/NQ/YM/RTY`

核心驱动：
- `FOMC / CPI / NFP / PCE`
- 美债收益率、美元、`VIX`
- 权重股财报与盘后指引
- 大盘广度与风险偏好

执行规则：
- 盘前和盘后 headline 风险很高，先确认时间段
- `NQ` 对利率和 mega-cap 财报更敏感
- `RTY` 更能反映广泛风险偏好和利率压力
- 宏观数据公布前后的 `1-2` 小时，技术位容易失真

## 8. 方向与止盈止损

先判断是否有方向优势：
- `做多`
- `做空`
- `观望`

只有 1-7 维中至少 4 维同向、关键维度无缺口、反向审计没有同等级证据时，才允许输出做多/做空。若结构、宏观、库存/CFTC、波动率或可执行合约层证据分裂，最终方向必须写观望，只给触发条件。

止盈止损规则：
- `SL` 锚定最近有效结构位，而不是固定百分比
- `TP1/TP2/TP3` 分别对应最近阻力/支撑、下一级结构位、扩展目标
- 至少给 `2` 个手动失效条件

常见失效条件：
- 关键周期收盘重新站回/跌回结构内部
- 核心波动率代理反向急变
- 重大消息面与原方向相反

## 9. 输出骨架

```markdown
## [标的] 交易判断

分析时间（UTC）: ...
周期: 日内 / 波段 / 中线
方向: 做多 / 做空 / 观望

一句话理由: ...
主导驱动: [按权重列 2-4 个]
最近轨迹: [趋势推进 / 箱体洗盘 / 冲高派发 / 跌破回收 / 阴跌磨人]
结构位: [支撑 / 阻力 / 摆点时间]

交易计划:
- 入场: ...
- SL: ... [结构依据]
- TP1: ... [结构依据]
- TP2: ... [结构依据]

失效条件:
1. ...
2. ...
```

## 10. 反向方向快速检查

当用户追问另一边是否可行时，不要把整篇分析重写一遍。

使用这个格式：

```text
做多需要: [条件列表]
当前: [逐条对照]
结论: [一句总结]
```
