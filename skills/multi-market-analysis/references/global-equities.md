# Global Equities

用于美股、全球 ETF、全球现金指数，以及无法直接归入商品期货或加密的股票类标的。

## 1. 先分清交易语境

| 标的 | 默认分析语境 |
|------|-------------|
| 美股个股 | 公司事件 + 大盘环境 + 结构 |
| ETF | 上游指数/行业/资产驱动 + 结构 |
| 全球现金指数 | 宏观偏向与节奏，不默认给精确交易点 |
| A股 | 不在本文件内展开，转用 A 股 canonical skill |

## 2. 美股个股

优先级：
- 公司硬新闻、财报、指引、监管
- 所属行业和龙头联动
- 利率、美元、大盘风险偏好
- 技术结构

执行规则：
- 短线分析必须检查财报窗口
- 如果财报在 `3` 个交易日内，且没有足够事件定价信息，默认降级为 `观望` 或保守偏向
- 盘前/盘后价格波动不能直接等价于常规时段流动性
- 对高贝塔成长股，要把 `QQQ`、收益率、美元一起看

## 3. ETF

ETF 不要只看图：

- `SPY` / `VOO`：先看美股大盘、收益率、`VIX`
- `QQQ`：先看 mega-cap 科技和利率
- `IWM`：先看小盘风险偏好和融资环境
- `XLE`：先看油价
- `GLD`：先看黄金驱动，不要直接当成普通股票
- `USO`：先看原油期货驱动和展期问题
- `TLT`：先看长端利率与宏观预期

如果用户分析的是 ETF，而真正驱动来自另一资产，就要写出“上游驱动先于 ETF 图形”。

## 4. 全球现金指数

例如：
- `^GSPC`
- `^IXIC`
- `^DJI`
- `^HSI`
- `^N225`

默认输出：
- `偏多`
- `偏空`
- `震荡`

只有在用户明确要对应的可交易载体时，才切换到期货/ETF/差价合约语境。

## 5. A股路由

A股不要沿用美股框架。转到以下 canonical skill：

- `/root/.hermes/skills/research/a-share-market-analysis/SKILL.md`
- `/root/.hermes/skills/research/a-share-market-analysis/references/china-finance-apis.md`

A股必须额外注意：
- `T+1`
- 午间休市
- 节假日和政策面
- 北向资金、涨跌家数、板块轮动

默认更适合输出：
- `偏多`
- `偏空`
- `震荡`

## 6. 推荐输出格式

### 美股个股 / ETF

```markdown
分析时间（UTC）: ...
周期: 日内 / 波段 / 中线
结论: 做多 / 做空 / 观望

核心理由: ...
公司/上游驱动: ...
最近轨迹: ...
结构位: ...
风险事件: 财报 / 指引 / 宏观数据 / 无
```

### 全球现金指数 / A股指数偏向

```markdown
分析时间（UTC）: ...
周期: 次日预判 / 波段 / 中线
结论: 偏多 / 偏空 / 震荡

主导因子: ...
最近轨迹: ...
关键位: ...
限制: [T+1 / 休市 / 数据缺口]
```
