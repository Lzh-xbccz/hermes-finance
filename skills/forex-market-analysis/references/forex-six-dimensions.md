# Forex Seven-Dimension Evidence

把前七维证据采集框架迁移到外汇，第八维 CZSC 只做确认/冲突/不足：

## 1. 七维映射

| 维度 | 外汇对应项 | 说明 |
|------|-----------|------|
| 技术面 | 多周期结构、支撑阻力、波段节奏 | 保留 |
| 市场结构 | 利差预期、即期与美元指数关系、避险/风险偏好结构 | 替代链上 |
| 主导力量 | 央行路径、收益率差、美元强弱、政策预期 | 替代 OI/费率 |
| 情绪面 | `VIX`、避险流、市场 risk-on/risk-off | 保留为替代 |
| 宏观基本面 | CPI、NFP、GDP、PMI、央行会议、地缘 | 权重高 |
| 交叉验证 | `DXY`、美债收益率、相关交叉盘、黄金/原油等关联资产 | 替代多交易所 |
| 方向门槛 | 同向证据数、关键缺口、最强反证 | 决定能否给做多/做空 |

## 2. 数据源建议

- `Yahoo Finance`
  - `EURUSD=X` `JPY=X` `GBPUSD=X` `AUDUSD=X` `USDCNY=X` `DX-Y.NYB`
- `Reuters`
  - 宏观 headline、央行、数据公布、政策新闻
- `CME FedWatch` 或 Reuters 对 Fed 定价解读
  - 作为利率路径背景
- `^TNX` / `^FVX`
  - 收益率代理
- `^VIX`
  - 风险偏好

## 3. 货币对的主导逻辑

### `EURUSD`
- Fed 路径 vs ECB 路径
- `DXY`
- 美德收益率差或欧债风险叙事

### `USDJPY`
- 美债收益率
- 日银政策与干预风险
- 风险偏好与避险流

### `GBPUSD`
- 英国通胀与央行路径
- 美元方向

### `AUDUSD` / `NZDUSD`
- 中国与全球风险偏好
- 大宗商品情绪
- 美元

### `USDCNH`
- 中国政策与中美利差
- 人民币管理与离岸波动
- 风险事件

## 4. 方向规则

先判断是否有方向优势：
- `做多`
- `做空`
- `观望`

注意：
- 方向是针对用户给的交易对本身，而不是某个货币的主观看法
- 例如看涨美元兑日元，对 `USDJPY` 是 `做多`
- 只按独立维度计票，不能把 DXY、10Y、USD 利差拆成多票；只有技术结构、美元/利差/利率、风险情绪、CFTC/仓位等至少 3 个独立维度同向，且领先反向至少 2 个维度、反向审计不构成同等级证据时，才允许做多/做空

## 5. 默认观望场景

- 重大数据公布前
- 结构与央行/收益率方向冲突
- 美元、利差、风险偏好与技术方向分裂
- 多空证据接近或关键利率/DXY 数据缺失
- 当前价格贴近关键位导致盈亏比差
