# US Equity Seven-Dimension Evidence

把前七维证据采集框架迁移到美股时，关键区别是公司事件和利率环境。第八维 CZSC 只做确认/冲突/不足。

## 1. 七维映射

| 维度 | 美股对应项 | 说明 |
|------|-----------|------|
| 技术面 | 结构、支撑阻力、量价、缺口 | 保留 |
| 市场结构 | 行业轮动、ETF 资金流、盘前盘后状态、广度 | 替代链上 |
| 主导力量 | 财报、指引、回购、机构配置、利率敏感性 | 替代 OI/费率 |
| 情绪面 | `VIX`、热度、风险偏好、涨跌家数 | 保留为替代 |
| 宏观基本面 | Fed、收益率、美元、经济数据、监管 | 权重高 |
| 交叉验证 | 行业 ETF、指数、收益率、同类龙头 | 替代多交易所 |
| 方向门槛 | 同向证据数、关键缺口、最强反证 | 决定能否给做多/做空或偏多/偏空 |

## 2. 数据源建议

- `Yahoo Finance`
  - 个股、ETF、指数价格与历史
- `Reuters`
  - 财报、公司 headline、监管、宏观
- `^VIX`
- `^TNX`
- `SPY` `QQQ` `IWM` `XLF` `XLK` `XLE` 等相关 ETF

## 3. 个股

重点顺序：
- 财报与指引
- 行业龙头联动
- 利率和大盘风险偏好
- 技术结构

默认观望：
- 财报在 `3` 个交易日内且没有清晰事件定价
- headline 风险过大
- 公司事件代理缺失但技术面单独偏多/偏空
- 大盘、行业 ETF、VIX、利率与个股技术方向分裂

## 4. ETF

先看上游驱动：
- `SPY` -> 标普大盘、收益率、`VIX`
- `QQQ` -> 大型科技、收益率
- `IWM` -> 小盘融资环境
- `XLE` -> 油价
- `GLD` -> 黄金
- `TLT` -> 长端利率

## 5. 现金指数

对 `^GSPC` `^IXIC` `^DJI` 这类现金指数，默认更适合输出：
- `偏多`
- `偏空`
- `震荡`

若用户明确要可执行载体，则转成 ETF 或期货语境。
